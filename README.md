## Instalación y Uso

### Instalar dependencias:

```bash
npm install
```

### Iniciar el servidor de desarrollo:

```bash
npm start
```

### Abrir en el navegador:

Visita [http://localhost:4231](http://localhost:4231) para ver el sitio en funcionamiento.
